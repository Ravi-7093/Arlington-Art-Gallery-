1) Install php from the given location - https://www.php.net/downloads.
2) Install xampp from the given location - https://www.apachefriends.org/index.html.
3) Ensure the database set up as per phase 2 of the project. (Import the given sql file into database to set up the art_gallery database)
4) Place all the script (.php) and html files inside the htdocs folder which is under the XAMPP folder(the place where it is stored on user machine).
5) Check the connection to db by verifying the database names in php files (databasename in php file should be same as database running in user machine).
6) Start the database servers by clicking on the start .
7) Enter the url as mentioned- http://localhost/art_gallery/.
8) Click on the buttons to navigate to the results.
